Website Evalutations

Lings Cars

This is a website for Lings car leasing service based out of the UK. Apparently it is purposely crazy to attract attention.

Crap Principles

  • Contrast: Elements that aren’t the same should be very different so they stand out and are easy to read.
  • Repition: Repeating visual elements of the design throughout the piece.
  • Alignment: All elements should line up and be organized.
  • Proximity: Elements that are related should be grouped together, whereas separate design elements
    should have enough space in between to communicate they are different.

Evaluation

This website has so many different colors and text sizes and a crazy colorful background, so contrast is not properly followed.
Everywhere you look, there are different colors, sizes, and formats. Repition is not followed. There is some, but very little consistent alignment.
There is whitespace in the middle that is random and does not contribute to proximity. Other than that, everything is so jumbled together
that there is no meaningful proximity. The developer desperatley needs to refer to the principles listed above.

Ford M-Sport Raptor T1+

This is a website for awwwawards.com, which is a design competition. This website shows the daily winner, as well as nominees, courses, and more.

Crap Principles

  • Contrast: Elements that aren’t the same should be very different so they stand out and are easy to read.
  • Repition: Repeating visual elements of the design throughout the piece.
  • Alignment: All elements should line up and be organized.
  • Proximity: Elements that are related should be grouped together, whereas separate design elements
    should have enough space in between to communicate they are different.

Evaluation

The background is minimalist, and makes the text pop out. Texts are very different but consistent sizes, contrast is well designed.
All titles/subtitles, image/video sizes, captions, etc. have consistent sizing respective to the section or type of text that it is.
Colors are few and repeated, sections are aligned, titles are aligned, content within sections is all perfectly aligned.
There is meaningful–but not too much–white space. Content within sections are close in proximity. This designer clearly knows their CRAP.
